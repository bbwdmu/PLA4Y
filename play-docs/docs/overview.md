# PLA4Y Documentation

PLA4Y is a modular platformer toolkit for Unreal Engine 5.7


It is designed to provide reusable gameplay systems with minimal dependencies, allowing developers to add only the features they need.

## Core Modules

- Collectibles
- Doors and progression
- Abilities and power-ups
- Health and lives
- Mascot and companion systems
- UI support
- Save system

## Documentation Structure

This documentation is split into setup guides, system breakdowns, Blueprint flow references, and technical design notes.
