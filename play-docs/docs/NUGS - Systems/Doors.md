# Door System

Controls progression using collectibles or abilities.

---

## Components

- BP_BaseDoor  
- AC_DoorLock  

---

## Features

- Collectible requirements  
- Optional consumption  
- Ability-based unlocking  

---

## Flow

1. Player interacts  
2. Requirements checked  
3. Door opens if valid  

---

## Usage

Attach DoorLock component and define requirements.
